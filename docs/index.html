Neural Radiance Fields for Dynamic Scenes

SIGGRAPH Asia 2021 · Park et al.

Abstract

We present Nerfies, a method for modeling dynamic scenes using neural radiance fields. Our method reconstructs both geometry and appearance over time, enabling novel-view synthesis of dynamic subjects. We achieve this by learning a scene representation that includes time as an input and propose a deformation network to model motion.
